###  Place cdn
```
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@10"></script>`
```

### Call Swal function on methods
```
Swal.fire({
    title: 'Error!',
    text: 'Do you want to continue',
    icon: 'error',
    confirmButtonText: 'Cool'
})
```
